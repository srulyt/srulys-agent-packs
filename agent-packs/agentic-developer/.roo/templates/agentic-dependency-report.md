# Dependency Report Template

Use this template to generate `.agent-memory/runs/<run-id>/dependency-report.md`.

---

## Template Content

```markdown
# Dependency Report

**Run ID**: {{RUN_ID}}
**Generated**: {{TIMESTAMP}}
**Platform**: {{PLATFORM}}

## Summary

| Status | Count |
|--------|-------|
| ✅ Installed | {{INSTALLED_COUNT}} |
| ⚠️ Outdated | {{OUTDATED_COUNT}} |
| ❌ Missing | {{MISSING_COUNT}} |
| **Total Required** | {{TOTAL_COUNT}} |

## Detected Technology Stack

Based on PRD analysis:
{{#each TECHNOLOGIES}}
- {{this}}
{{/each}}

## Tool Status

| Tool | Required | Detected Version | Minimum Version | Status |
|------|----------|------------------|-----------------|--------|
{{#each TOOLS}}
| {{name}} | {{#if required}}Yes{{else}}No{{/if}} | {{detected_version}} | {{min_version}} | {{status_icon}} {{status}} |
{{/each}}

## Actions Required

{{#if HAS_MISSING}}
### Missing Tools

The following tools must be installed before proceeding:

{{#each MISSING_TOOLS}}
- **{{name}}**: {{description}}
{{/each}}

### Installation Script

A platform-specific installation script has been generated:

- **Script**: `{{SCRIPT_PATH}}`
- **Platform**: {{PLATFORM}}

To install all missing dependencies, run:

{{#if IS_WINDOWS}}
```powershell
.\install-dependencies.ps1
```
{{else}}
```bash
chmod +x install-dependencies.sh
./install-dependencies.sh
```
{{/if}}

After running the script, verify installations and re-run the dependency check.
{{else}}
✅ All required tools are installed and meet minimum version requirements.

No action required. Ready to proceed with bootstrap planning.
{{/if}}

## Version Details

{{#each TOOLS}}
### {{name}}

- **Command**: `{{detection_command}}`
- **Output**: `{{raw_output}}`
- **Parsed Version**: {{detected_version}}
- **Required**: {{min_version}}
- **Status**: {{status}}

{{/each}}

---

*Generated by Agentic Developer - Bootstrap Planner*
```

---

## Field Definitions

| Field | Description | Example |
|-------|-------------|---------|
| `RUN_ID` | Current run identifier | `2026-01-24-a1b2c3d4` |
| `TIMESTAMP` | ISO-8601 generation time | `2026-01-24T15:30:00Z` |
| `PLATFORM` | Detected platform | `Windows 11`, `macOS Sonoma`, `Ubuntu 22.04` |
| `INSTALLED_COUNT` | Number of tools installed | `5` |
| `OUTDATED_COUNT` | Number of outdated tools | `1` |
| `MISSING_COUNT` | Number of missing tools | `2` |
| `TOTAL_COUNT` | Total required tools | `8` |
| `TECHNOLOGIES` | List of detected tech from PRD | `["Node.js", "React", "Docker"]` |
| `TOOLS` | Array of tool status objects | See below |
| `HAS_MISSING` | Boolean: any missing/outdated | `true` |
| `SCRIPT_PATH` | Path to install script | `install-dependencies.ps1` |
| `IS_WINDOWS` | Boolean: Windows platform | `true` |

### Tool Object Structure

```json
{
  "name": "Node.js",
  "required": true,
  "detected_version": "18.17.0",
  "min_version": "18.0",
  "status": "Installed",
  "status_icon": "✅",
  "detection_command": "node --version",
  "raw_output": "v18.17.0",
  "description": "JavaScript runtime for building web applications"
}
```

### Status Icons

| Status | Icon |
|--------|------|

| Outdated | ⚠️ |
| Missing | ❌ |

## Usage Notes

1. **Replace placeholders**: All `{{FIELD}}` values must be replaced with actual data
2. **Conditional sections**: `{{#if}}` and `{{#each}}` are for documentation; use string manipulation in actual implementation
3. **Simple string replacement**: This pack uses simple find-replace, NOT Handlebars templating
4. **Preserve formatting**: Maintain table alignment and code block syntax
