# Customer Story Deck Example

A *customer-voice* deck used by AEs in a second sales call. The prospect
should leave the call thinking about the reference customer (Northwind),
not the vendor.

## Files

- `context.md` — paste into `@story-orchestrator`.
- `expected-deck-shape.json` — golden shape for the
  `smoke-customer-story` eval case (if added).

## What to look for

- Design system should be `customer-coral` (warm, customer-forward).
- Hero's Journey framing: setup → trial → resolution.
- Every metric is *introduced by* or *paired with* a Northwind quote.
- The CTA is one ask: "agree to a 6-week paid pilot."
