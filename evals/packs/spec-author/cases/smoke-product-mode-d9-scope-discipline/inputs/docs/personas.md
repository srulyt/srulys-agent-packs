# personas.md (fixture)

## Product Manager (PM)

Owns the product roadmap. Wants a daily summary so they can spot
emerging activity patterns without reading every changelog entry.
Cares about: freshness (≤ 24h), opt-out controls, signal quality.

## Engineering Manager (EM)

Owns delivery health. Wants a daily summary that surfaces
cross-team blockers and unusual spikes in change volume. Cares
about: completeness, cross-team scope, drill-through to source.
